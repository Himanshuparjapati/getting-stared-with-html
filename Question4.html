Html tag

  • Tags are the fundamental components of HTML and are enclosed in angle brackets (< >).
  • Tags define the structure and content of different elements on a web page.
  • HTML tags are often used in pairs, with an opening tag and a closing tag, to surround content.

HTML Element

  • An HTML element is a complete set of tags and the content they enclose.
  • Elements can be categorized into two types: inline and block-level.

  • Inline Element

    hese elements do not start on a new line and only take up as much width as necessary. Examples include <span> <strong> and <a> .
  • Block-level Elements:: These elements start on a new line and take up the full width available. Examples include <div>, <p>, and <h1> to <h6> (headings). Elements can have attributes that provide additional information about the element.
